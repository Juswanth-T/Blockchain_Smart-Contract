# Blockchain Smart Contract

A Solidity smart contract built during blockchain development coursework at IIT Madras. Demonstrates core Ethereum smart contract concepts including state management, transactions, and event emission.

## Contents

The contract source is packaged in `Smart Contract.zip`.

```bash
unzip "Smart Contract.zip"
```

## About

This repository contains a Solidity smart contract implementing fundamental blockchain application patterns. Written as an introductory exercise in decentralized application (dApp) development.

## Tech Stack

- Solidity
- Ethereum (EVM)
- Remix IDE

## Getting Started

1. Extract the ZIP file
2. Open the `.sol` file in [Remix IDE](https://remix.ethereum.org/)
3. Select the appropriate Solidity compiler version
4. Deploy to a test network (Sepolia, Goerli) or local instance (Ganache / Hardhat)
